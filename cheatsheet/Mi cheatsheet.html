Mi CheatSheet

Etiquetas básicas

<html>

Define el documento HTML.

<html lang="es">
    <head>
        <title>Título de la página</title>
    </head>
    <body>
        <p>Contenido de la página</p>
    </body>
</html>

<head>

Contiene metadatos y enlaces a recursos externos.

<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>

<body>

Contiene el contenido visible de la página web.

<body>
    <p>Contenido de la página</p>
</body>

Contenido de la página

Estructura semántica

<header>

Define el encabezado de una sección o página.

<header>
    <h1>Bienvenido a Mi Sitio</h1>
</header>

Bienvenido a Mi Sitio

<nav>

Define una sección de navegación.

<nav>
    <ul>
        <li><a href="#home">Inicio</a></li>
        <li><a href="#about">Acerda de...</a></li>
    </ul>
</nav>
  • Inicio
  • Acerca de...

<main>

Define el contenido principal del documento.

<main>
    <p>Contenido principal aquí.</p>
</main>

Contenido principal aquí.

<aside>

Define contenido aparte del contenido principal, como barras laterales.

<aside>
    <p>Esto es un contenido adicional.</p>
</aside>

Esto es un contenido adicional.

<footer>

Define el pie de página de una sección o documento.

<footer>
    <p>© 2024 by Jonathan-Serrano</p>
</footer>

Listas

<ol>

Define una lista ordenada.

<ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ol>
  1. Elemento 1
  2. Elemento 2

<ul>

Define una lista desordenada.

<ul>
    <li>Elemento A</li>
    <li>Elemento B</li>
</ul>
  • Elemento A
  • Elemento B

Define una lista de definiciones.

    <dd>Lenguaje de marcado para crear páginas web.</dd>
</dl>

Formularios

<form>

Define un formulario HTML para la entrada del usuario.

<form action="/submit" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

<input>

Define un control de entrada en un formulario.

<input type="text" name="nombre">

<select>

Define una lista desplegable.

<select name="opciones">
    <option value="1">Opción 1</option>
    <option value="2">Opción 2</option>
</select>

<textarea>

Define un área de texto.

<textarea name="mensaje">
    Escribe tu mensaje aquí.
</textarea>

<label>

Define una etiqueta para un control de formulario.

<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">

Variantes del elemento <input>

type="text"

Define un campo de texto de una sola línea.

<input type="text" name="nombre">

type="number"

Define un campo para números.

<input type="number" name="edad">

type="tel"

Define un campo para números de teléfono.

<input type="tel" name="telefono">

type="email"

Define un campo para direcciones de correo electrónico.

<input type="email" name="email">

type="password"

Define un campo para contraseñas.

<input type="password" name="contrasena">

type="radio"

Define un botón de opción.

<input type="radio" name="genero" value="masculino"> Masculino
<input type="radio" name="genero" value="femenino"> Femenino
Masculino Femenino

type="checkbox"

Define una casilla de verificación.

<input type="checkbox" name="aceptar"> Acepto los términos y condiciones
Acepto los términos y condiciones

type="button"

Define un botón que no envía el formulario.

<input type="button" value="Haz clic">

type="color"

Define un control para seleccionar un color.

<input type="color" name="color favorito">

type="date"

Define un control para seleccionar una fecha.

<input type="date" name="fecha nacimiento">

type="time"

Define un control para seleccionar una hora.

<input type="time" name="hora">

type="datetime-local"

Define un control para seleccionar una fecha y hora.

<input type="datetime-local" name="fecha y hora">

type="file"

Define un control para subir archivos.

<input type="file" name="archivo">

type="hidden"

Define un campo oculto.

<input type="hidden" name="id_usuario" value="123">
